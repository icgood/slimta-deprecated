Modules

slimta The root module for all other slimta modules.
slimta.base64 Module containing encoding and decoding functions for the standard BASE64 codec, as provided by the OpenSSL crypto library.
slimta.hmac Module for generating HMAC-encoded digests using OpenSSL's HMAC() function.
slimta.rlimit Module providing access to the setrlimit and getrlimit system calls.
slimta.signalfd Module for catching signals in an event-based manner.
slimta.uuid Module for generating "Universally Unique IDentifiers".
slimta.xml Module that provides an interface to XML parsing using libexpat.
