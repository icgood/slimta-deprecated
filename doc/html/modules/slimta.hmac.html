Module slimta.hmac

encode (method, data, key)
Encodes the data using the given secret key. This is not a reversible action.

Parameters

  • method: the method string. The available methods are the corresponding EVP_*() names, e.g. "md5" or "sha1".
  • data: a string of data to encode.
  • key: a secret key to encode the data with.

Return value:

a string of bytes containing the result of the encoding.
