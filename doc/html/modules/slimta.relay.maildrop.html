Module slimta.relay.maildrop

Module that implements a relayer using the "maildrop" command-line utility. This is a venerable, local delivery method often used in conjunction with courier-POP or mutt.

Functions

slimta.relay.maildrop.new (cmd, time_limit) Creates a new maildrop relayer.


Functions

slimta.relay.maildrop.new (cmd, time_limit)
Creates a new maildrop relayer.

Parameters

  • cmd: the command name, default "maildrop".
  • time_limit: maximum time to run maildrop before failing the message.

Return value:

a new maildrop relayer object.
