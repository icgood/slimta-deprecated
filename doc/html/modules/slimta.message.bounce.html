Module slimta.message.bounce

Module for taking a message and response and generating a bounce message to explain to the sender why the original message failed to send.

Functions

Functions

slimta.message.bounce.build (self, message, response, timestamp)
Takes information about the original message and what happened to it and generates a bounce message to alert the sender that the message failed to be delivered and why.

Parameters

  • self: the bounce generator object.
  • message: the original message object.
  • response: the message response object.
  • timestamp: timestamp to use for the bounce message, defaulting to the current time.
slimta.message.bounce.new (sender, client, header_tmpl, footer_tmpl)
Creates a new bounce generator object. A form of variable substitution is applied to the header_tmpl and footer_tmpl strings allowing them to contain information about the message, see the usage manual for more details.

Parameters

  • sender: used as the envelope sender of the bounce message, default "".
  • client: if given, used as the message.client of the bounce message.
  • header_tmpl: The MIME data preceding the original message contents.
  • footer_tmpl: The MIME data proceeding the original message contents.

Return value:

a new bounce generator object.
