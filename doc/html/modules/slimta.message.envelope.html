Module slimta.message.envelope

Module for describing the envelope of a message, in other words the info used by policies and the relayer to deliver the message.

Functions

slimta.message.envelope.copy (old)
Copies a message envelope object into a new object. This is a "deep" copy operation, tables are not recycled.

Parameters

  • old: the original message envelope object.

Return value:

a new message envelope object, identical to the original.
slimta.message.envelope.from_meta (meta)
Loads a message envelope object from the given simple table of meta information.

Parameters

  • meta: simple table of meta information to load from.

Return value:

a new message envelope object.
slimta.message.envelope.from_xml (tree_node)
Loads a message envelope object from the given XML node tree, as generated by the slimta.xml.reader module.

Return value:

slimta.message.envelope.new (sender, recipients, dest_relayer, dest_host, dest_port)
Creates a new message envelope object from the given information.

Parameters

  • sender: the original sender address of the message.
  • recipients: table array of intended message recipients.
  • dest_relayer: string identifying the intended relayer.
  • dest_host: string identifying the destination host.
  • dest_port: the destination port number.

Return value:

a new message envelope object.
slimta.message.envelope.to_meta (self, meta)
Creates a simple key-value table describing all the information contained in the message envelope object, suitable for serialization or, more commonly, storage in a storage engine.

Parameters

  • self: the message envelope object.
  • meta: if given, keys are added to this table rather than a new one.

Return value:

the simple table of meta information.

Generates a table array of XML representing the message envelope object.

Parameters

  • self: the message envelope object.

Return value:

a table array of XML lines.
