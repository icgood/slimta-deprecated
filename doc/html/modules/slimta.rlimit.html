Module slimta.rlimit

Module providing access to the setrlimit and getrlimit system calls. In server applications, tuning these values often proves valuable. The rlimit resource parameter in the get() and set() functions below correspond to RLIMIT_* constants, e.g. "CPU" and "NOFILE".

Functions

get (resource) Gets the current soft and hard rlimit values, respecticaly.
set (resource, soft, hard) Sets new soft and hard rlimit values.
